架构说明

数据流向：Spring Boot + SkyWalking Agent → gRPC → SkyWalking OAP Server → SkyWalking UI

Agent 启动：-javaagent:/path/to/skywalking-agent.jar -Dskywalking.agent.service_name=my-app

追踪注解：@Trace 标记方法、@Tag 标记参数、ActiveSpan.tag() 自定义标签

SkyWalking UI：http://localhost:8080 查看服务拓扑、链路详情、指标仪表盘

日志关联：TraceIdPatternLogbackLayout 自动在日志中注入 traceId
