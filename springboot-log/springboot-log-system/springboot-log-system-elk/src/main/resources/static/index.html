ELK 日志系统测试

Elasticsearch + Logstash + Kibana | 日志通过 LogstashTcpSocketAppender 发送到 Logstash

1. 创建订单（正常业务流程）

2. 日志级别测试

输出 TRACE/DEBUG/INFO/WARN/ERROR 各级别日志，在 Kibana 中可按级别筛选查看。

3. 异常日志测试

模拟除零异常，错误日志及堆栈信息将通过 Logstash 收集到 Elasticsearch。

架构说明

数据流向：Spring Boot App → LogstashTcpSocketAppender → Logstash:5044 → Elasticsearch → Kibana

日志格式：JSON（logstash-logback-encoder 自动将日志转为结构化 JSON）

Kibana 查看：http://localhost:5601 创建 Index Pattern 后在 Discover 中搜索日志

关键配置：logback-spring.xml 中的 LogstashTcpSocketAppender 配置 Logstash 地址
