架构说明

数据流向：Spring Boot App → JSON 日志文件/stdout → Fluentd → Elasticsearch → Kibana

日志格式：JSON（logstash-logback-encoder 输出结构化 JSON 到文件）

适用场景：K8s 环境中 Fluentd 以 DaemonSet 部署，自动采集容器 stdout 日志

对比 ELK：EFK 用 Fluentd 替代 Logstash，更轻量、K8s 原生支持
