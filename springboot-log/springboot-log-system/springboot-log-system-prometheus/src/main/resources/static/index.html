Prometheus 指标监控测试

Spring Boot Actuator + Micrometer | Counter / Gauge / Timer / DistributionSummary 四种指标类型

1. 模拟 API 调用（Counter + Timer）

2. 模拟错误（Counter）

触发错误计数器 +1，可在 Prometheus 中查看 api_errors_total 指标。

3. 在线用户管理（Gauge）

Gauge 指标随用户上下线实时增减，反映当前在线人数。

4. Actuator 端点

Prometheus 指标 健康检查 所有端点

架构说明

数据流向：Spring Boot (Micrometer) → /actuator/prometheus → Prometheus Server 定期抓取 → Grafana Dashboard

指标类型：Counter（累计值）、Gauge（实时值）、Timer（耗时分布）、DistributionSummary（数据分布）

Prometheus：http://localhost:9090 配置 scrape target 为应用的 /actuator/prometheus

Grafana：http://localhost:3000 导入 Spring Boot 监控面板（ID: 4701 或 12900）
