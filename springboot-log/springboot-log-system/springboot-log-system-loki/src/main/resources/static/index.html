架构说明

数据流向：Spring Boot App → Loki4jAppender (HTTP) → Loki → Grafana Explore

查询语言：LogQL（类似 PromQL 的日志查询语言）

Grafana：http://localhost:3000 添加 Loki 数据源后在 Explore 中查询

优势：不依赖 Elasticsearch，存储成本低，与 Grafana 生态深度集成
