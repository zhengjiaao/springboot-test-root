📋 认证流程

1
用户提交用户名密码，调用 POST /api/auth/login
2
服务端验证后返回 AccessToken（30分钟）+ RefreshToken（7天），RefreshToken存入Redis
3
前端每次请求携带 Authorization: Bearer <accessToken>
4
AccessToken过期时，用RefreshToken调用 POST /api/auth/refresh 获取新双Token（轮转机制）
5
登出时AccessToken加入Redis黑名单 + 删除RefreshToken，确保Token立即失效
