📋 认证流程

1
用户提交用户名密码，调用 POST /api/auth/login
2
服务端验证后创建 HttpSession，返回 JSESSIONID Cookie
3
浏览器后续请求自动携带 Cookie: JSESSIONID=xxx，无需手动管理
4
勾选"记住我"后，额外颁发 remember-me Cookie（7天有效期）
5
同一用户仅允许1个Session，新登录会踢掉旧Session
6
登出时清除Session和Cookie，确保立即失效
